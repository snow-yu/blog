shell

shell 就是一个程序，接受从键盘输入的命令，然后把命令传递给操作系统去执行。
bash(Bourne Again SHell)：最初Unix上由Steve Bourne写成的shell程序sh的增强版

文件系统

linux 以分层目录结构来组织所有文件，即所有文件组成来一棵树目录（有且只有一个）。文件系统中的第一级目录称为根目录。根目录包含文件和子目录。

文件名规则

  1. 以“.”字符开头的文件名是隐藏文件
  2. 文件名和命令名是大小写敏感的
  3. linux没有文件扩展名的概念。文件内容或用途由其他方法来决定
  4. 文件名命名支持空格、标点符号（只限使用“.”，“-”，下划线），不要在文件名中使用空格

路径

  • 绝对路径

绝对路径开始于根目录（用“/”表示），紧跟着目录树的一个个分支，一直到达所期望的目录或文件。

  • 相对路径

相对路径开始于工作目录，文件系统中用两个特殊符号来表示相对位置。这两个特殊符号分别是“.”(工作目录)、“..”（工作目录的父目录）。

  • 使用建议

哪个输入量少就用哪个

特殊字符

  • “～”表示用户家目录

命令

命令 完整命令 选项 描述
pwd print working directory 显示当前工作目录
ls list directory contents 列出目录内容
-l 以长模式输入
-a 列出所有文件，包括被隐藏的隐藏文件
-d 展示目录，与-l集合使用查看制定目录的详细信息
-h(human readable) 以长格式列出时，以人们可读的格式
-r(reverse) 以相反的顺序来显示结果
-S(size) 按照文件大小来排序
-t(time) 按照修改时间来排序
cd change directory 更改工作目录
- 更改工作目录到先前的工作目录

linux结构

目录 描述
/ 根目录
/bin 包含系统启动和运行所必须的二进制程序
/boot 包含linux内核、初始RAM磁盘映象（用于启动时所需的驱动）和启动加载程序。
/dev 包含设备节点的特殊目录
/etc 包含所有系统层面的配置文件，也包含一系列的shell脚本
/home 用户权限当前目录
/lib 包含核心系统程序所使用的共享库文件
/lost+found 格式化分区或设备
/media 现在的Linux系统中，包含可移动介质的挂载点，比如usb设备、光盘等
/mnt 早期的Linux系统中，包含可移动介质的挂载点
/opt 包含可能安装在系统中的商业软件产品
/root root账户的家目录
/sbin 包含系统二进制文件，通常为超级用户保留
/temp 包含由各种程序创建的临时文件的地方
/usr 包含普通用户所需要的所有程序和文件
/usr/bin 包含系统安装的可执行程序，通常这个目录包含许多程序
/usr/lib 包含由/usr/bin 目录中的程序所用的共享库
/usr/local 由非系统发行版自带程序的安装目录
/usr/sbin 包含许多系统管理程序
/usr/share 包含许多由/usr/bin目录中的程序使用的共享数据
/usr/share/doc 大多数安装在系统中的软件会包含一些文档
/var 存放的是动态文件。各种数据库，用户邮件等，都放在这里
/var/log 包含日志文件、各种系统活动的目录
